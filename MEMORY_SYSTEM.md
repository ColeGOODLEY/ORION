# ORION MEMORY SYSTEM

## Purpose

The ORION Memory System stores important information that allows ORION to provide personalized guidance, track progress, and improve decision-making over time.

---

# Memory Categories

## 1. Operator Profile

Purpose:
Store information about ORION's operator.

Includes:
- Goals
- Strengths
- Weaknesses
- Interests
- Skills
- Preferences

---

## 2. Missions

Purpose:
Track important objectives.

Each mission contains:

Name:
Goal:
Deadline:
Progress:
Next Action:

---

## 3. Knowledge Base

Purpose:
Store information learned by the operator.

Includes:
- Books
- Lessons
- Skills
- Research
- Important concepts

---

## 4. Projects

Purpose:
Track active projects.

Each project contains:

Name:
Purpose:
Current Status:
Next Steps:
Challenges:

---

## 5. Network

Purpose:
Track important relationships.

Includes:
- People
- Companies
- Opportunities
- Follow-ups
- Notes

---

## 6. Performance Data

Purpose:
Track improvement over time.

Includes:
- Habits
- Productivity
- Health
- Financial progress
- Skills developed

---

# Memory Rules

ORION should:

1. Prioritize important information.
2. Avoid storing unnecessary information.
3. Use memories to provide better decisions.
4. Continuously improve understanding of its operator.
